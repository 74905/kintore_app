# ジムトモ

## サイト概要
よく行くジムの仲間を探して、共に高めあいながらトレーニングをする為のコミュニケーションサイト

### サイトテーマ
ジムの仲間を探す

### テーマを選んだ理由
昨今健康は謳われてはいるが、いざジムに行ってもなかなか続かない、もしくわ行ったとしても、器具の使い方がよくわかっていない人が多い為、
このサイトを作り、ジム内でコミュニケーションをとってもらい、質のあるトレーニングをしてもらう


### ターゲットユーザ
ジムに行く人全て

### 主な利用シーン
トレーニング中、もしくはトレーニングに行く前の道中
